Полезные утилитки для облегчения жизни WEB-программистов и кодировщиков.

Класс HTML
Этот класс предназначен для:
  • чтения и расшифровки/раскодирования HTML-файлов
  • генерации HTML-файлов
  • просмотра файлов в HTML-формате, возможно в последствии будет сделан полноценная WWW-бродилка, но пока на данный момент скорее всего будет применяться для HELP-системы и т.п. (смотрелка еще не реализована)
  • использования в серверных скриптах типа mod_perl,mod_java,... mod_clip
  • использования в CGI-BIN,CGI-WIN скриптах
  • использования в www_clip скиптах (об этом немного позже/ниже)
  • не хотелось бы забегать вперед, но чтобы дать полное представление об этом классе - возможно когда-нибудь этот класс станет основой для clip_script модуля в популярных www-бродилках.
  • и даже может быть на его базе когда-нибудь будет создан редактор HTML-документов.

Класс htmlParser
Этот класс предназначен для разбора html-документа на составляющие его теги, а теги на поля. Возвращает значения и объекты, которые без преобразования используются в объекте HTML (читайте чуть выше).
