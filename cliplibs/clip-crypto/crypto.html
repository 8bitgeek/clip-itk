Simmetric block ciphers encrypt/decrypt functions

EVP_ENCRYPT

EVP_ENCRYPT(cStr, cKey [, [cCipherMethod='des-ede3-cbc'] [, [cDigestMethod='md5'] [, [cInitVector='']]]]) --> cEncrypted

EVP_DECRYPT

EVP_DECRYPT(cStr, cKey [, [cCipherMethod='des-ede3-cbc'] [, [cDigestMethod='md5'] [, [cInitVector='']]]]) --> cEncrypted

some possible cipher names:
"DESX" "desx" "DES" "des" "DES3" "des3" "IDEA" "idea" "RC2" "rc2" "BF" "bf" "blowfish" "CAST" "cast" "CAST-cbc" "cast-cbc" "rc5" "RC5" "DSS1" "dss1" "ripemd" "rmd160"

some possible digest names:
"md5" "md4" "md2" "sha" "sha1" "dss" "dss1" "mdc2" "ripemd160"

EVP_ALG_LIST

EVP_ALG_LIST() -> aAlgoritmes

List algorithms, builtined in used version of OpenSSL library.
