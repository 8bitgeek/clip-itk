  VERSION()
$CATEGORY$
  SYSTEM,MISCELLANEOUS,INFO
$SUMMARY$
   Returns xClipper version

$SYNTAX$
     VERSION() --> cVersion
$LANG_RU$
     VERSION() --> номер версии

     EXTEND.LIB, as a character value.

     VERSION() возвращает номер версии xClipper библиотеки EXTEND.LIB в
     виде символьной строки.
$DESCRIPTION$
     VERSION() is an environment function that returns the version of the
     xClipper library, EXTEND.LIB.
$LANG_RU$
     VERSION() - функция среды, которая возвращает номер версии xClipper
     библиотеки EXTEND.LIB.
