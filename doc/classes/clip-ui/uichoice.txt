$ABOUT$
	Edit field with choice button. Inherits fields and method from class <link linkend="classuiedit">UIEdit</link>.
$LANG_RU$
	Поле ввода с кнопкой выбора элемента. Унаследуетс свойства класса <link linkend="classuiedit">UIEdit</link>.
$SYNTAX$
	UIChoice(<action>,[<value>],[<text>])	--> UIChoice object
$LANG_RU$
	UIChoice(<action>,[<value>],[<text>])	--> Объект UIChoice
$ATTRIBUTES$
	<className>	- Class name. Value should be "UIChoice".
	<button>	- <link linkend="classuibutton">UIButton</link> object. Button for select element.
	<source>	- String. Address of source to get text description for set value. 
$LANG_RU$
	<className>	- Имя класса. Значение: "UIChoice".
	<button>	- Объект <link linkend="classuibutton">UIButton</link>. Кнопка выбора элемента.
	<source>	- Строка. Адрес ресурса для получения текстового описания устанавливаемого значения.

	<value>		- Любой тип. Значение, хранимое в виджете.
	<text>		- Строка. Текст в поле ввода.
$ENDMETHOD$

$METHODNAME$
	setSource()
$SUMMARY$
	Set address of source to get text description for set value.
$LANG_RU$
	Установить адрес ресурса для получения текстового описания устанавливаемого значения.
$SYNTAX$
	setSource(<source>)
$ARGUMENTS$
	<source>	- String. Source address in form '[db:]class[:field]'. This address pass to 
	function getAttributeValue(source,value) (if exists) for fill text description of set value.
$LANG_RU$
	<source>	- Строка. Адрес источника в виде: '[db:]class[:field]'. Этот адрес будет передан 
	функции getAttributeValue(source,value) (если она есть) для заполнения строки показа 
	текстового описания.
$ENDMETHOD$
